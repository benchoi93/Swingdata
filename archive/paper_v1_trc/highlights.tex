\begin{highlights}
\item Analysis of 2.78M e-scooter trips with speed profiles across 52 Korean cities
\item Speed governor removal increases speeding odds 121-fold (OR = 121)
\item Rider experience increases speeding via progressive TUB mode adoption
\item SHAP analysis (AUC=0.905) validates TUB mode as dominant predictor
\item Curvy roads: lower speeding prevalence but 5x higher risk scores
\end{highlights}
