## Issues Found

1. **[MINOR] Factual error in TASKS.md age-survival narrative (line 176)**
   - TASKS.md states: "Age <20 longest survival (34 trips), 40+ second longest (37 trips)."
   - By the JSON data (`survival_analysis_results.json`), Age 40+ has median survival of 37 trips and Age <20 has 34 trips. Since 37 > 34, Age 40+ has the longest survival, not Age <20.
   - The daily report's table correctly shows both values (34 and 37) and the narrative correctly focuses on the Cox HR (Age <20 HR=0.85, protective), so the error is confined to the TASKS.md description only.
   - Suggested fix: Change TASKS.md line 176 to read "Age 40+ longest survival (37 trips), <20 second longest (34 trips)."

2. **[MINOR] No proportional hazards assumption check for Cox PH models**
   - The survival analysis script (`src/survival_analysis.py`) fits both `CoxPHFitter` (static) and `CoxTimeVaryingFitter` but does not verify the proportional hazards assumption via Schoenfeld residuals or `cph.check_assumptions()`.
   - For a dataset of 200K users, the PH assumption may be violated for some covariates (especially `ever_tub` which may have time-varying effects captured by the separate time-varying model).
   - Suggested fix: Add `cph_static.check_assumptions(model_df, p_value_threshold=0.05)` after fitting. If PH is violated for `ever_tub`, note this in the results JSON and mention that the time-varying model addresses this limitation. This is especially important for paper integration (Task 12.1.7).

3. **[MINOR] Mediation percentage methodology could be more rigorous**
   - The mediation analysis uses a simple difference-of-differences approach: (overall effect from first to last experience bin) minus (STD/ECO-only effect from first to last bin), then computes the proportion. This is the Baron-Kenny approach, which provides a reasonable point estimate but does not account for confounders or provide a confidence interval.
   - For the paper, consider the Sobel test or bootstrap mediation CI to strengthen the 98.4% claim.
   - Suggested fix: Either add a bootstrap CI to the mediation estimate, or clearly note the Baron-Kenny limitation when integrating into the paper. Given that the mediation proportion is 98.4% with only 0.38pp direct effect, even a wide CI would not change the qualitative conclusion, so a brief methodological caveat may suffice.

4. **[MINOR] Report KM table incomplete for event counts by TUB/age**
   - The daily report's KM table shows "-" for n_events in the TUB users, Non-TUB users, and all age group rows. The JSON contains these values (e.g., TUB users: 295,517 events, Non-TUB: 118,992 events). Including them would improve completeness.
   - Suggested fix: Fill in the event counts in the report table for future reference.

## Recommendations

- **Proportional hazards check before paper integration.** This is the most important methodological gap. Run `cph.check_assumptions()` and report the Schoenfeld residual test results. If PH is violated for `ever_tub`, frame the time-varying model as the preferred specification and the static model as a sensitivity check.

- **Consider Aalen additive hazards as a robustness check.** The Aalen model does not require the PH assumption and can reveal time-varying covariate effects nonparametrically. With lifelines 0.30.3, `AalenAdditiveFitter` is available. This would strengthen the robustness argument.

- **Bootstrap the mediation percentage.** Even a simple nonparametric bootstrap (resample users 1000 times, recompute mediation %) would provide a CI around the 98.4% estimate. Given the large sample size (864K users), the CI will be tight, but it demonstrates rigor for TR-C reviewers.

- **Clarify the "5.4x faster" claim in the report.** The report states "TUB users reach first speeding 5.4x faster than non-TUB users" (81/15 = 5.4). This is the ratio of medians, which is interpretable but technically the hazard ratio (HR=2.85 from static Cox, or 10.41 from time-varying) is the standard measure of "how much faster." Consider specifying "median survival time is 5.4x shorter" to avoid conflating with the HR.

- **For paper integration (Task 12.1.7):** Position the survival analysis as addressing the endogeneity concern raised in the Discussion's Limitations section. The time-varying Cox model (HR=10.41) provides within-person evidence that is stronger than the cross-sectional ever_tub HR=2.85, complementing the DiD analysis which is at the city-aggregate level.

## Suggested Research Directions

### 1. **Competing Risks Survival Model for Speeding Severity Tiers** (Feasibility: HIGH)
   - **Method**: Fine-Gray subdistribution hazard model or cause-specific hazard model, treating first-speeding as competing events with different severity thresholds: (a) mild speeding (25-30 km/h), (b) moderate speeding (30-35 km/h), (c) severe speeding (>35 km/h). The current binary event (any speeding) collapses meaningful heterogeneity. A competing risks framework would reveal whether TUB mode accelerates progression to *severe* speeding specifically, or merely to mild threshold violations.
   - **Data needed**: Already available. The `speeds` column in `routes_all.parquet` contains the full speed vector; `max_speed_from_speeds` provides the trip maximum. Classify each speeding trip into severity tiers and identify the first occurrence of each tier per user. Use the same Feb-Nov 2023 window and user survival dataset from Phase 12.
   - **Expected finding**: TUB adoption disproportionately accelerates time-to-first-*severe*-speeding (HR > 15 for >35 km/h tier), while mild speeding onset is less mode-dependent. This would sharpen the policy argument: governors do not merely reduce threshold violations but specifically prevent the highest-risk behavior.
   - **Target venue**: Transportation Research Part C (extends the current paper) or Accident Analysis & Prevention (methodological contribution to micromobility risk assessment).
   - **Connection to current work**: Directly extends the Phase 12 Cox PH framework. The 98.4% mediation finding applies to binary speeding; competing risks would decompose this mediation by severity tier. The existing threshold sensitivity analysis (Phase 5.7: 20/25/30 km/h) provides context but is cross-sectional; competing risks adds the longitudinal dimension.
   - **Key reference**: [Keramati (2020) -- Competing Risks for Highway-Rail Grade Crossings](https://onlinelibrary.wiley.com/doi/10.1155/2020/8878911) applied this framework to transportation safety with multiple severity outcomes.

### 2. **Deep Learning on Raw Speed Sequences for Risky Trip Classification** (Feasibility: HIGH)
   - **Method**: Train a 1D Temporal Convolutional Network (TCN) or Transformer encoder on the raw per-trip speed vectors (variable-length sequences of 5-200+ readings at ~10s intervals) to classify trips as safe, mildly risky, or severely risky. The current modeling pipeline extracts hand-crafted features (mean, max, P85, CV, speeding rate, cruise fraction) and discards the sequential structure. A sequence model could discover latent speed patterns (acceleration bursts before speeding, oscillatory "testing-the-limit" profiles) that summary statistics miss.
   - **Data needed**: The `speeds` column is already parsed and available. Use the May 2023 modeling dataset (2.78M trips with labels) for training. The 4-class GMM labels (Safe, Moderate-Risk, Habitual Speeder, Stop-and-Go) serve as ground truth for a multi-class classification task. Alternatively, define risk labels from the hurdle model (Part 1: speeder/non-speeder, Part 2: severity quantiles).
   - **Expected finding**: The sequence model achieves AUC > 0.92 (exceeding the LightGBM AUC=0.905 from SHAP analysis) and identifies temporal patterns such as "gradual acceleration in trip first half predicts speeding in second half" (consistent with the -1.5 km/h speed drift finding from Phase 8.2). Attention weights would highlight the critical time windows within a trip where intervention (e.g., real-time speed warning) would be most effective.
   - **Target venue**: NeurIPS Workshop on AI for Transportation or TR Part C (if combined with policy simulation showing how real-time intervention based on early-trip detection could reduce speeding by X%).
   - **Connection to current work**: Builds on the SHAP analysis (Phase 8.4) by moving from feature importance to sequence-level prediction. Aligns with the PI's expertise in deep generative models for transportation (TrajGAIL) and the dataset's unique strength of having full within-trip speed profiles at scale. The [European Transport Research Review paper on cycling anomaly detection with CNNs](https://etrr.springeropen.com/articles/10.1186/s12544-023-00583-4) provides a methodological template for micromobility sequence classification.

### 3. **Causal Mediation Analysis of TUB Mode Adoption as Mechanism** (Feasibility: MEDIUM)
   - **Method**: Formal causal mediation analysis using the potential outcomes framework (Imai, Keele, Tingley 2010). Treatment: rider experience (operationalized as log trip rank). Mediator: TUB mode adoption (binary: ever adopted TUB by trip *t*). Outcome: time-to-first-speeding. Estimate the Average Causal Mediation Effect (ACME) and Average Direct Effect (ADE) with sensitivity analysis for unmeasured confounding (rho parameter). This replaces the informal Baron-Kenny mediation (98.4%) with a formally identified causal quantity.
   - **Data needed**: The survival dataset from Phase 12 already contains all required variables (`trip_rank`, `first_tub_rank`, `first_speed_rank`, `event_time`). Need to restructure as a sequential ignorability setup where experience affects TUB adoption, which in turn affects speeding onset. The DiD analysis (Phase 11) provides exogenous variation in TUB availability that can serve as an instrument or identification check.
   - **Expected finding**: ACME accounts for 90-99% of the total experience effect, consistent with the 98.4% Baron-Kenny estimate, but now with a formal confidence interval and sensitivity bounds. The sensitivity analysis would show that an unmeasured confounder would need to explain >80% of the residual variation to reduce the mediated proportion below 50%, confirming robustness.
   - **Target venue**: Transportation Science (methodological rigor in causal identification) or Journal of the American Statistical Association Application section (novel application of causal mediation to transportation safety).
   - **Connection to current work**: Directly formalizes the Phase 12 mediation finding. The current 98.4% estimate is compelling but methodologically informal. A formal causal mediation analysis would address the most likely TR-C reviewer concern: "How do you know TUB adoption is a mediator and not a collider or a consequence of unobserved risk preference?" The sensitivity analysis would quantify exactly how strong a confounder would need to be to overturn the finding. The [Cicchino et al. (2024) study on speed limiters and rider behavior](https://journals.sagepub.com/doi/10.1177/03611981231214518) provides a natural comparison: their cross-city design cannot disentangle selection from treatment, but the within-user longitudinal data here can.
