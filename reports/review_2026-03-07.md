## Issues Found

1. **[MINOR]** JSON data type inconsistency in `mode_switcher_placebo_results.json`
   - The `placebo_passes` field is stored as the string `"True"` rather than the boolean `true`. This occurs because `abs(did_effect) < abs(original_did)` may produce a numpy bool, which `json.dump` with `default=str` converts to a string instead of a native JSON boolean.
   - Suggested fix: Explicitly cast to Python bool before serialization: `'placebo_passes': bool(abs(did_effect) < abs(original_did))`.
   - Impact: None for the paper; only affects downstream consumers of the JSON file.

2. **[MINOR]** Sign convention inconsistency for Cohen's d in the report
   - The session summary states "d = 0.030" (absolute value, no sign), while the table correctly reports "Cohen's d = -0.030 (negligible)". The discussion.tex uses "$d = 0.03$" (positive, no sign).
   - Suggested fix: Adopt a consistent convention -- either always report absolute values with direction noted separately, or always include the sign. For the paper, using absolute value with direction noted in prose (as currently done in discussion.tex) is acceptable for TR-C style.
   - Impact: Cosmetic only; the magnitude and interpretation are correct throughout.

3. **[MINOR]** Hardcoded `original_did = 0.2013` in `mode_switcher_placebo.py` (line 221)
   - The treatment DiD is hardcoded rather than read from `mode_switcher_results.json`. If the original analysis is ever re-run with different filters, the comparison would be stale.
   - Suggested fix: Load the value from `mode_switcher_results.json` at runtime.
   - Impact: None currently; only a maintainability concern.

The placebo speed change difference is statistically significant (t = -5.37, p < 0.001) but negligibly small (d = -0.030), which actually strengthens the argument: large samples can detect statistical significance even when practical significance is absent.

## Suggested Research Directions

1. **Within-Trip Speed Sequence Classification Using Transformers** (Feasibility: HIGH)
   - Method: Train a 1D Transformer or temporal convolutional network on raw per-trip speed sequences (variable-length time series of ~10s intervals) to classify trips as safe vs. risky. Use the 25 km/h threshold to label trips, then learn latent speed patterns that predict risk beyond simple summary statistics (mean, max, CV). Apply attention visualization to identify which temporal segments of a trip are most predictive.
   - Data needed: `speeds` column from `routes_all.parquet` (already available); road class sequences from `trip_road_classes.parquet` as auxiliary features. All 44.1M trips available, though a stratified subsample of 1-2M would suffice for training.
   - Expected finding: The model should achieve AUC > 0.90 (comparable to the LightGBM SHAP model at 0.905) but additionally reveal that acceleration patterns in the first 30 seconds of a trip are highly predictive of eventual speeding -- potentially enabling real-time intervention systems. The attention weights would provide interpretability beyond SHAP, showing *when* during a trip risk emerges.
   - Target venue: Transportation Research Part C (methodological contribution) or NeurIPS workshop on ML for transportation
   - Connection to current work: Extends the SHAP analysis (Model 8) from aggregate feature importance to temporal dynamics within trips. The current paper treats speed profiles as summary statistics; this direction exploits the full sequential structure. Aligns with PI's expertise in deep generative models for trajectories (TrajGAIL). Recent work on trajectory prediction using transformers (e.g., [TrajLearn 2025](https://dl.acm.org/doi/full/10.1145/3729226)) has focused on spatial prediction; applying similar architectures to speed safety classification would be novel.

2. **Spatial Difference-in-Differences: Geofencing Policy Simulation** (Feasibility: HIGH)
   - Method: Combine the city-level DiD framework (Model 9) with the H3 hex-level spatial analysis to simulate targeted geofencing policies. Use the dose-response relationship (5.0 pp speeding reduction per 10 pp TUB reduction) to estimate counterfactual outcomes if TUB were banned only in hot-spot hexes (19% of Seoul). Apply spatial Durbin DiD to test whether speed reductions in treated hexes spill over to adjacent untreated hexes. This exploits the fact that post-December 2023 data shows near-complete TUB removal (natural experiment for full ban) while the pre-period shows hex-level variation in TUB usage.
   - Data needed: `data_parquet/spatial/Seoul.parquet` (H3 hex-level aggregates, already computed), `city_month_panel.parquet` (already available), and a hex-month panel constructed from `routes_all.parquet` by aggregating trips to H3 hexes by month.
   - Expected finding: Spatial spillover coefficient is positive -- speed reductions in treated hexes propagate to adjacent hexes, amplifying the direct effect by 15-30%. This would provide the first empirical evidence for geofencing spillover effects in micromobility, directly informing the paper's geofencing policy recommendation. The simulation would quantify the cost-benefit of targeted vs. blanket bans.
   - Target venue: Transportation Research Part C or Transportation Science (methodological and policy contribution)
   - Connection to current work: Bridges the gap between the aggregate DiD (Section 5.9) and the spatial analysis (Section 5.2). The current paper establishes both spatial clustering (Moran's I) and causal effects (DiD) separately; this direction integrates them. The Cicchino et al. (2024) study on [speed-limiter settings and behavior](https://journals.sagepub.com/doi/10.1177/03611981231214518) examined between-city variation but not within-city spatial targeting. Recent policy scholarship on [evidence-based e-scooter regulation](https://www.sciencedirect.com/science/article/pii/S0967070X24000982) calls for more granular spatial policy tools, which this analysis would directly provide.

3. **Survival Analysis for Time-to-First-Speeding with Mode Adoption as Time-Varying Covariate** (Feasibility: MEDIUM)
   - Method: Fit a Cox proportional hazards model (or an accelerated failure time model) where the event is a user's first speeding violation (max speed > 25 km/h) and survival time is measured in cumulative trips. Include TUB mode adoption as a time-varying covariate (binary: has the user adopted TUB by trip k?), controlling for age, city, and time-of-day composition. Test whether TUB adoption triggers speeding onset or whether riders who would eventually speed simply adopt TUB sooner (joint frailty model for mode adoption and speeding onset).
   - Data needed: `data_parquet/all_months/user_longitudinal.parquet` (1.82M users with longitudinal histories) and `data_parquet/modeling/trip_experience.parquet` (trip-level experience data). Requires constructing a per-trip survival dataset with time-varying TUB exposure indicator. The 60.9% of users who never speed provide natural censoring.
   - Expected finding: TUB adoption is the strongest time-varying predictor of speeding onset (hazard ratio > 5). Importantly, a joint frailty model should reveal that approximately 20-30% of the "experience effect" (OR = 1.178 per log-trip) is confounded by TUB adoption timing -- riders who adopt TUB early speed earlier, but the within-mode experience effect is modest. This would formalize the mediation finding from the GEE analysis (Section 5.7) using a more rigorous survival framework.
   - Target venue: Transportation Research Part C or Accident Analysis & Prevention
   - Connection to current work: The current paper establishes that experience increases speeding (GEE, OR = 1.178) and that this is mediated by mode adoption. Survival analysis would provide a more natural framework for this question -- "how long until a rider first speeds?" rather than "what fraction of trips involve speeding?" -- and directly address the policy question of graduated mode access timing. The 60.9% never-speeders provide a large censored population that logistic regression underutilizes but survival analysis handles naturally. The finding that TUB adoption rises from 11.5% at trip 1 to 47.8% at trip 50 suggests a time-varying covariate framework is essential.
