## Key Findings / Observations

### Critical Regression Interpretation Error
- The time-of-day regression text was **backwards**: claimed "elevated" evening/night speeding when ORs were 0.79/0.84 (lower than midday)
- Root cause: the text was written based on descriptive patterns (raw speeding rates are higher at night) without accounting for the confounding by mode composition (late-night riders disproportionately use TUB mode)
- This is a substantive error that would have been caught in peer review but is embarrassing to submit

### Filter Table Math Inconsistency
- Sequential running totals implied 52,328 trips removed, but actual unique exclusions were 49,571 (difference: 2,757 trips flagged by multiple filters)
- Fixed by removing misleading "Remaining" column and noting filter overlap
